# 07 Branch Admin

## SPEC

Build `/app/branch/*` for local students, teachers, classes, rooms, schedule, attendance, payments, reports, messages, and settings.

## PLAN

- Map branch scope to rooms, users, classes, invoices, attendance, and events.

## IMPLEMENT

- Build branch operations around local capacity, room status, class readiness, and schedule conflicts.

## VERIFY

- Create room, update room status, check branch-scoped staff/classes, test mobile.
- Run `scripts/verify.sh`.

## REVIEW

- UI reviewer.
- QA reviewer.

## FIX

Fix branch scope, room capacity, and schedule issues.

## DOCUMENT

Document branch-specific assumptions.
